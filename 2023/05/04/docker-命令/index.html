Docker 命令

安装 docker

curl -sSL https://get.daocloud.io/docker | sh

查看容器

sudo docker ps -a

查看镜像

sudo docker images

删除镜像

sudo docker rmi ID

删除容器

sudo docker remove ID

拉取镜像

sudo docker pull ubuntu:18.04

在指定镜像中创建容器

添加-it 参数交互运行 添加-d 参数后台运行

sudo docker run -dit –name ubuntu1 ubuntu:18.04 /bin/bash

（也可把 ubuntu:18.04 改成 image ID）

离开容器

exit || ctrl + D

启动容器

docker start 容器ID/容器名称

进入容器

docker exec -it 容器ID/容器名称 /bin/bash

宿主机文件拷贝到容器内

sudo docker cp mengxuegu mycentos2:/opt

导入本地镜像

cat tina-lidar-bt.tar.gz | docker import - r16

停止运行中的容器

docker stop myubuntu

重启容器

docker restart myubuntu
